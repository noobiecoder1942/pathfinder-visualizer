# Pathfinder Visualizer

A web app to visualize several Single-Source Shortest Path (SSSP) algorithms.

## Local development

`pnpm install` then `pnpm run dev`

## Tech stacks

- React (Typescript) using Vite
- Javascript for algorithms
- Sass for styling
